# c-cpp-kata-template
Kata template for C and C++ using Cest Framework

# Requirements
- CMake 3.5 or higher
- A C/C++ compiler

# Running the tests

Simply run:

```
make
```

# Coding

Add C code to `example.c` and C++ code to `example.cpp`. Respective headers are called `c_exports.h` and `cpp_exports.h`. Test file is `example.test.cpp`
